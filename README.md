#### To run this project locally:

1. Click Fork on GitHub

2. Clone your fork:

```
git clone https://github.com/rhw-repo/tailwind_project.git

```

3. Navigate into the project directory:

```
cd tailwind_project
```

4. Install dependencies:

```
npm install
```

or

```
yarn install
```

5. Start the dev server

```
npm run dev
```

or

```
yarn dev
```
